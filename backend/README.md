# React-Store-Backend

## Pre-Requisites
* NodeJS
* MySQL 5.7+ / MariaDB 10.1.29

## Getting Start
1. Execute SQL in
[/#DatabaseObject](https://github.com/arbaieffendi/React-Store-Backend/tree/master/%23DatabaseObject)
2. Run MySQL
3. Clone this project
4. Install all dependency in package.json
`npm install`
5. Start the server
`nodemon`
